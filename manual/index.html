1 시스템 개요

악티보 일정관리 시스템은 감리/제안 프로젝트의 단계·투입인력·일정을 통합 관리하는 웹 애플리케이션입니다. 여러 사용자가 동시에 접속하여 작업할 수 있으며, 동시 수정 충돌을 방지하는 점유 잠금 기능을 제공합니다.

2 로그인

시스템은 OIDC(OpenID Connect) 기반 SSO 로그인을 사용합니다. 별도 비밀번호 없이 조직 계정으로 로그인합니다.

1
로그인 페이지에 접속하면 "로그인" 버튼이 표시됩니다.
2
버튼 클릭 → OIDC 인증 화면으로 이동하여 조직 계정으로 로그인합니다.
3
인증 완료 후 홈 화면으로 자동 이동합니다.
4
관리자가 허용 사용자 목록에 등록해야 접근이 가능합니다. 미등록 시 접근 거부 화면이 표시됩니다.
⚠️ 접근 불가 시 관리자에게 계정 등록을 요청하세요. 관리자 페이지(/admin)에서 이메일 또는 표시 이름으로 허용 사용자를 등록할 수 있습니다.

4-1 프로젝트 생성

"+ 새 프로젝트" 버튼 클릭 시 생성 다이얼로그가 열립니다. 상태에 따라 입력 방식이 다릅니다.

기본 입력 필드

필드필수설명
프로젝트명필수사업 이름. 공백 불가.
기관명필수발주 기관명. 공백 불가.
상태—감리 (기본값) 또는 제안 선택
비고—자유 메모. 입력 선택사항.

감리 모드 — 단계/투입공수 일괄 입력

단계, 날짜, 인력을 한 번에 텍스트로 입력하면 단계·투입공수·기본 일정이 자동 생성됩니다 (선택사항).

# 형식: 단계명, 시작일(YYYYMMDD), 종료일(YYYYMMDD), 인력1:분야[:MD], 인력2:분야[:MD], ... 요구정의, 20250224, 20250228, 이현우:사업관리 및 품질보증, 차판용:응용시스템 개략설계, 20250421, 20250430, 이현우:사업관리 및 품질보증, 강진욱:SW개발보안:4
요소형식설명
단계명자유 텍스트쉼표(,) 제외. 예: 요구정의, 개략설계
시작일 / 종료일YYYYMMDD8자리 날짜. 예: 20250224
인력:분야이름:분야MD 미지정 시 단계 전체 영업일 자동 계산
인력:분야:MD이름:분야:숫자MD 직접 지정. 예: 강진욱:SW개발보안:4
💡 분야 → 팀 자동 분류
  • 사업관리 포함 → 단계감리팀 (사업관리 및 품질보증 순서 0)
  • 응용시스템 포함 → 단계감리팀 (순서 1)
  • 데이터베이스 포함 → 단계감리팀 (순서 2)
  • 시스템구조 포함 → 단계감리팀 (순서 3)
  • 위 패턴 외 → 전문가팀으로 분류

제안 모드 — 제안서 일정/인력 입력

상태를 제안으로 선택하면 입력 화면이 분리된 섹션으로 바뀝니다.

📅 감리 일정 입력

# 형식: 단계명, YYYYMMDD, YYYYMMDD, 이름A, 이름B:3 # 이름만 = 전체 기간, 이름:숫자 = MD 지정 설계-정밀진단, 20260323, 20260327, 강혁, 김현선, 최규택:3 설계-재검증, 20260427, 20260501, 강혁, 김현선

👤 인력 섹션 입력

# 형식: 이름, 분야 (한 줄에 한 명) -- 감리원 섹션 -- 강혁, 사업관리 및 품질보증 김현선, 응용시스템 -- 핵심기술 섹션 -- 최규택, 핵심기술
섹션category 저장값설명
👤 감리원단계감리팀단계감리 인력. 이름, 분야 형식
🔹 전문가 - 핵심기술핵심기술핵심기술 전문가
🔹 전문가 - 필수기술필수기술필수기술 전문가
🔹 전문가 - 보안진단보안진단보안진단 전문가
🔹 전문가 - 테스트테스트기능테스트 전문가
✅ 섹션 분류 동작 방식 각 섹션에 입력된 이름은 일정 텍스트 등장 여부와 무관하게 해당 category로 정확히 저장됩니다. 예를 들어 핵심기술 섹션에 입력한 인력은 일정에 없어도 category=핵심기술로 저장됩니다.

4-4 투입인력 관리

각 단계에 투입될 인력과 투입공수(MD)를 관리합니다.

투입인력 추가

1
단계 행에서 "인력 추가" 버튼 클릭
2
인력 검색 드롭다운에서 이름/팀/등급으로 검색 후 선택
또는 "외부 인력" 선택 후 이름 직접 입력
3
분야(field), 투입공수(MD) 입력 후 저장

투입인력 입력 규칙

필드필수조건
인력필수DB 등록 인력 선택 또는 외부 인력 이름 직접 입력
분야 (field)필수자유 텍스트. 팀 분류에 사용됨
투입공수 (MD)필수숫자. 단계 영업일 수를 초과하면 경고 표시
category—자동 설정 (감리 모드: 분야 패턴 기반 / 제안 모드: 섹션 기반)
⚠️ MD 초과 입력 주의 투입공수(MD)가 단계 영업일 수를 초과하면 경고가 표시됩니다. 저장은 가능하나 일정 표에서 표시 불일치가 발생할 수 있습니다.

5 인력 관리

상단 탭의 👥 인력에서 인력을 등록·조회합니다.

인력 등록

"+ 새 인력 등록" 버튼 클릭 후 다이얼로그에서 입력합니다.

필드필수설명
이름필수한글 이름. 공백 불가. 중복 등록 시 인력별 일정에서 구분이 어려울 수 있음.
직급—예: 수석, 책임, 선임 등
감리원 등급—감리 자격 등급. 예: 특급, 고급, 중급
구분—재직 상태 또는 소속 구분. 자유 입력.

인력 상세 (/person/:id)

인력 이름을 클릭하면 해당 인력의 상세 페이지로 이동합니다. 이름·직급·등급·구분 수정과 개인 일정을 확인할 수 있습니다.

투입인력인력 교체, 분야, MD 수정 / 인력 추가(삭제)
단계감리팀 / 전문가팀으로 그룹 구분
💡 MD 변경 시 달력 자동 확장 안내 투입공수(MD)를 늘리면 "기존 MD로 유지 / 전체 확장" 선택 다이얼로그가 표시됩니다.
  • 기존 MD로 유지: 달력 변경 없이 MD 수만 변경
  • 전체 확장: 늘어난 영업일만큼 달력에 자동 추가

동시 접속 잠금 — 주간별 사업 일정

9 텍스트로 단계 편집 (일괄 입력)

🔴 주의 — 전체 덮어쓰기 "전체 덮어쓰기" 저장 시 기존 모든 단계·투입공수·일정 데이터가 삭제되고 입력한 텍스트 기반으로 재생성됩니다. 실행 전 확인 팝업이 표시됩니다.

  • 최규택:3 → 해당 단계 중 3일만 투입

인력 섹션 (감리원/전문가) 입력 형식
